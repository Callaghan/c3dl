C3DL Canvas Resize Test

This page tests the resizing ability of the canvas. Enter a width and height into the appropriate boxes and click the 'Resize' button. The canvas and the content it displays should resize to the requested scale. Changing the height of the canvas does not change what is visble in the scene, but scales it according to the new size of the canvas. An object that was half the height of the canvas before resizing will be half the height of the canvas afterwards. Width does not change the display of the objects in the scene. Instead it changes how much of the scene you can see horizontally. Note that as you get further from a square canvas objects directly in front of the camera will appear as normal, but objects on the periphery (extreme left and right) will be significantly distorted.
